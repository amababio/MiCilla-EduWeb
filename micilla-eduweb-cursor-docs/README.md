# MiCilla EduWeb Cursor Documentation Pack — Batch 1

This folder contains the first Cursor implementation documents for MiCilla EduWeb.

## Files

1. `00-project-rules.md` — Main rules Cursor must follow.
2. `01-master-implementation-roadmap.md` — Full phased roadmap.
3. `02-phase-0-project-setup.md` — Step-by-step Phase 0 setup instructions.
4. `03-phase-1-public-demo-website.md` — Step-by-step Phase 1 public demo website instructions.
5. `19-testing-checklist.md` — Testing checklist for all phases.
6. `20-common-cursor-prompts.md` — Ready-to-use Cursor prompts.
7. `21-change-control-and-scope-rules.md` — Scope protection and feature control rules.

## Recommended Use

Start Cursor with:

```txt
Read 00-project-rules.md first and follow it strictly.
Do not implement anything yet.
Confirm your understanding of the project direction, current phase, allowed scope, and stop point.
```

Then paste:

1. `01-master-implementation-roadmap.md`
2. `02-phase-0-project-setup.md`

After Phase 0 is complete and tested, proceed to:

3. `03-phase-1-public-demo-website.md`

Do not start database, admin dashboard, or backend work until Phase 1 is reviewed and approved.
